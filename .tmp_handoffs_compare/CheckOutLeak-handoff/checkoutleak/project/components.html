03 · Primitives

03Components

Eighteen primitives. They are sharp, small, and built to recede. The interesting thing on any screen is the data, not the chrome that holds it.

C · 01
Buttons
Four intents: primary, secondary, ghost, danger. Primary is a luminous neutral, not a brand color — this keeps the signal amber sacred. Danger is the only severity-colored button.
C · 02
Text inputs, selects, toggles
Low-contrast well, hairline border, amber focus. Help text is mono — even when it's prose — to keep the form surface visually quiet.

C · 03
Pills & badges
Always mono, always uppercase, always small. Pills name states; they never name CTAs. Severity pills use the severity palette; status pills use the semantic palette.

C · 05
Panels
Vault uses panels, not cards. A panel is a surface shift plus a hairline border — never a shadow, never a gradient. Adjacent panels share borders rather than creating two separate edges.
