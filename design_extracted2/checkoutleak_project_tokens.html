01 · Foundation

01Tokens

Every value in Vault resolves to a token. If it isn't a token, it doesn't ship. Tokens use OKLCH for perceptual consistency and are exposed as CSS custom properties.

01.1

01.2
Signal — the amber rule
One accent. It marks money at risk. It is reserved: not for CTAs, not for links, not for active navigation. Breaking this rule breaks trust in the system.
signal
oklch(0.78 0.13 75)
Foreground for money-at-risk numerals, severity bars on high/critical, and a single pixel-thin underline on active nav.

signal-dim
signal @ 14% alpha

Accent stroke on issue rows that carry dollar amounts. Always left-edge, never full-border.

01.7
Borders — hairlines
Three weights, all 1px. Weight varies by alpha, not thickness. A 2px border is never correct.
line-subtleoklch(1 0 0 / 0.04) · dividers between rows
line-defaultoklch(1 0 0 / 0.07) · panels, inputs, groups
line-strongoklch(1 0 0 / 0.12) · hover, emphasized
01.8
Shadow — reserved
Shadows are a last resort. They appear only on floating layers: popovers and modals. Panels and cards do not cast shadows.
Popover — shadow-popover
Modal — shadow-modal
01.9
Motion
One easing curve. Four durations. Nothing overshoots. Nothing bounces. Active scans may pulse at 2s; nothing else animates on idle.
dur-instant80ms · micro-feedback
dur-fast120ms · hovers, state swaps
dur-base180ms · panel transitions
dur-slow280ms · modal open/close
easing · cubic-bezier(0.2, 0, 0, 1) — no rebound
01.10
Focus
Focus is a contract. Every interactive element has a visible focus ring in the signal color — yes, the same amber reserved for money. Focus is the one other thing the system treats with equal seriousness.
outline 2px signal · offset 2px
border + 3px signal @ 14%
Nav link
keyboard only · not on :hover
01.11
Layering
Five named layers. No z-index above 80 is ever correct.

01.12
In code
Every token is a CSS custom property on :root. Tailwind, styled-components, or vanilla — pick any, but consume the same variables.
/* tokens.css — excerpt */
:root {
  /* surface */
  --ink-050: oklch(0.17 0.008 250);
  --ink-100: oklch(0.19 0.008 250);

  /* signal — money at risk only */
  --signal: oklch(0.78 0.13 75);

  /* severity */
  --sev-critical: oklch(0.64 0.22 22);

  /* motion — one curve to rule them all */
  --ease: cubic-bezier(0.2, 0, 0, 1);
  --dur-fast: 120ms;
}
